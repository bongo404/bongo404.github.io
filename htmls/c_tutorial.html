Welcome to C Programming for Beginners! This tutorial will guide you through the fundamentals of C, one of the most influential programming languages ever created. You'll learn everything from basic syntax to building your first real program. No prior programming experience required.

Chapter 1: Getting Started

C is a powerful general-purpose programming language created by Dennis Ritchie in 1972. It's been the foundation for operating systems, embedded systems, and countless other applications. Learning C gives you a deep understanding of how computers work at a fundamental level.

Before we write code, you'll need to set up your development environment. For this tutorial, we recommend using GCC (GNU Compiler Collection), which is available on Windows, macOS, and Linux.

üì¶ Installation Guide

Windows: Install MinGW or use WSL (Windows Subsystem for Linux)

macOS: Install Xcode Command Line Tools: xcode-select --install

Linux: Install GCC: sudo apt-get install build-essential

Once installed, verify your setup by opening a terminal and typing:

gcc --version

You should see version information displayed. You're now ready to start programming!

Chapter 2: Your First Program

Let's write the classic "Hello, World!" program. Create a new file called hello.c and type the following code:

#include <stdio.h> int main() { printf("Hello, World!\n"); return 0; }

Let's break down what each part does:

#include <stdio.h> - This line includes the Standard Input/Output library, which gives us access to printf() and other I/O functions.

int main() - This is the main function where program execution begins. Every C program must have exactly one main() function.

printf() - This function prints text to the console. The \n creates a new line.

return 0; - This tells the operating system that the program finished successfully.

To compile and run your program:

gcc hello.c -o hello ./hello

Chapter 3: Variables & Data Types

Variables are containers that hold data. In C, you must declare variables with their type before using them. Here are the fundamental data types:

int age = 25; // Integer numbers float price = 19.99; // Decimal numbers char grade = 'A'; // Single characters double pi = 3.14159; // High-precision decimals

The type tells the compiler how much memory to allocate and how to interpret the data. Each type has different size and range limitations.

üìä Common Data Types

int: Typically 4 bytes, range -2,147,483,648 to 2,147,483,647

float: 4 bytes, ~6-7 decimal digits precision

double: 8 bytes, ~15-16 decimal digits precision

char: 1 byte, single ASCII character

Here's a complete example using different types:

#include <stdio.h> int main() { int score = 95; float temperature = 72.5; char initial = 'J'; printf("Score: %d\n", score); printf("Temperature: %.1f\n", temperature); printf("Initial: %c\n", initial); return 0; }

Chapter 4: Input & Output

So far we've only printed output. Now let's learn to get input from users using scanf():

#include <stdio.h> int main() { int age; printf("Enter your age: "); scanf("%d", &age); printf("You are %d years old.\n", age); return 0; }

The & symbol (address-of operator) is crucial - it tells scanf() where to store the input value in memory.

‚ö†Ô∏è Important Note

Always use & with scanf() for basic types (int, float, char). This is one of the most common mistakes beginners make!

Chapter 5: Control Flow

Control flow statements let your program make decisions and repeat actions. The if statement checks conditions:

#include <stdio.h> int main() { int age; printf("Enter your age: "); scanf("%d", &age); if (age >= 18) { printf("You are an adult.\n"); } else { printf("You are a minor.\n"); } return 0; }

For repeating actions, use loops. Here's a for loop that counts from 1 to 5:

for (int i = 1; i <= 5; i++) { printf("%d\n", i); }
