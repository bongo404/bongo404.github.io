How I Made a Server with an Old Laptop

Published on January 15, 2025 by Kristi

This project started when I looked at my old laptop and thought — why not turn this into something useful? That’s when I decided to make it a home server.

Why I Did This

As an IT student with an old laptop gathering dust, I wanted to learn more about servers and Linux, while also making that laptop useful again. Turning it into a personal home server was a perfect project.

Step 1: Choosing the OS

I decided to install Arch Linux for more control and learning opportunities. Arch gave me a clean base to build exactly what I needed, and it helped me understand how everything connects under the hood.

Step 2: Setting Up File Sharing with Samba

Once Arch was installed and configured, I set up Samba so I could share files across my home network. This allowed me to access files from other devices like my phone or desktop.


[shared]
   path = /home/kk/shared
   browsable = yes
   writable = yes
   guest ok = yes
   create mask = 0644
   directory mask = 0755

Step 3: Adding Security

I didn't want an open network share. I configured user authentication and added a ufw firewall to only allow specific traffic.


sudo ufw allow Samba
sudo ufw enable

Step 4: Hosting My Blog

Later, I used the same laptop to host my personal blog. I installed Nginx and pointed it to my HTML folder. Now, I can access my blog using my local IP or even share it publicly (with proper port forwarding).

What I Learned

  • Linux installation and package management
  • Samba configuration for file sharing
  • Firewall setup for basic security
  • Nginx web server hosting

Conclusion

Turning an old laptop into a home server is one of the best DIY projects for tech students. It’s cheap, practical, and packed with learning.
